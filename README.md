# 使用方法:
如果你正在使用https://github.com/risin42/NagramX
第三方开源版Telegram，是最方便直接导入的。

❶.打开NagramX，找到N-设置

❷.点击实验性功能，找到消息过滤器打开

❸.找到全局过滤器打开，然后导添加正则表达式
