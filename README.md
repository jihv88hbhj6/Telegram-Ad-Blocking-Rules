[长声明👈](https://github.com/jihv88hbhj6/statement)

这个项目我称他为“Telegram之清郎行动”
起因Telegram的广告和诈骗数不胜数，严重影响日常使用，所以我在业余时间找的大量广告关键词，尽可能的不影响日常使用体验（群聊，机器人功能等）
规则目前已超1千以上，只针对简体中文和少量繁体中文有效。

# 使用方法:
如果你正在使用<a href="https://github.com/NextAlone/Nagram" target="_blank">Nagram</a>
或<a href="https://github.com/risin42/NagramX">Nagram X</a>第三方开源的Telegram，是最方便直接导入的。（但我个人只用<a href="https://github.com/risin42/NagramX">Nagram X</a>🙈）


❶.先下载<a href="https://github.com/jihv88hbhj6/Telegram-Ad-Blocking-Rules/releases">屏蔽广告规则</a>
，到设备文件夹里

❷.打开NagramX，找到（N-设置）

❸.点击（实验性功能），找到（消息过滤器）打开

❹.找到（全局过滤器）打开，把<a href="https://github.com/jihv88hbhj6/Telegram-Ad-Blocking-Rules/releases">屏蔽广告规则</a>导入（添加正则表达式）即可完成全部步骤
